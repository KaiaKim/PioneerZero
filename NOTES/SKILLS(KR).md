# 스킬 목록 (Skill List)

* `P` 표기는 **필살기**를 의미하며, 전투당 기본 1회 사용 가능
* 모든 수치는 **기본값**이며, 최종 수치는 밸런싱 과정에서 조정 가능
* 공격력 보정, 우선도 보정은 **체인에 사용했을 때** 적용된다
